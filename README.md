# cs50p exercises

This repository contains solutions to the Harvard CS50P (Python) problem sets and exercises organized by week.

Layout:

- Week0/ ... Week8/: exercise files and tests (where provided)

Usage:

1. Clone the repo
2. Optionally create a virtual environment:

```bash
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt  # if you add dependencies
```

3. Run individual scripts directly, for example:

```bash
python3 Week0/einstein.py
```
